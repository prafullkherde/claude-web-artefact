## 🚀 Setup in 5 Steps (One Time Only)

### Step 1 — Create Repository
1. Go to **github.com/prafullkharde**
2. Click **"New repository"** (green button, top right)
3. Name it: `knowledge-hub`
4. Set to **Public**
5. Click **"Create repository"**

### Step 2 — Upload the File
1. In your new repo, click **"Add file" → "Upload files"**
2. Drag and drop **`MyKnowledgeHub.html`**
3. **IMPORTANT:** Rename it to **`index.html`** before uploading
   (GitHub Pages needs the main file to be called index.html)
4. Click **"Commit changes"**

### Step 3 — Enable GitHub Pages
1. Click **"Settings"** tab in your repo
2. Scroll down to **"Pages"** in left sidebar
3. Under **"Source"** → select **"Deploy from branch"**
4. Branch: **main** | Folder: **/ (root)**
5. Click **"Save"**

### Step 4 — Wait 2 Minutes
GitHub builds your site. Then visit:
```
https://prafullkharde.github.io/knowledge-hub/
```

### Step 5 — Done! 🎉
Bookmark that URL. It never changes. Always live.

## 🧠 Mental Model Format Used

Every learning topic follows the **P → E → S** model:
```
Problem:  What question does this answer?
Evidence: What data/facts support it?  
Solution: The insight or conclusion
```

This format builds mental models that actually stick,
not just notes that get forgotten.
